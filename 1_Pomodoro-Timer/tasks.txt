# Pomodoro Timer Project Tasks

## Project Setup
1. **Create Project Directory**
   - Open VS Code.
   - Create a new folder: `Pomodoro-Timer`.
   - Open this folder in VS Code.

2. **Initialize Project Files**
   - Inside `Pomodoro-Timer`, create the following files:
     - `index.html`
     - `style.css`
     - `script.js`

---

## Task 1: Build the HTML Structure
1. **Create the Timer Display**
   - Add a `<div>` to show the countdown timer (default: `25:00`).
   
2. **Add Control Buttons**
   - `Start` button
   - `Pause` button
   - `Reset` button

3. **Session Label**
   - Add a `<p>` tag to indicate `Work` or `Break` mode.

4. **Link CSS and JavaScript**
   - Connect `style.css` and `script.js` inside `index.html`.

---

## Task 2: Style the UI with CSS
1. **Design Timer Display**
   - Use a large font for the timer.
   - Center it on the page.
   
2. **Style Buttons**
   - Add padding, colors, and hover effects.
   
3. **Session Indicator**
   - Different colors for `Work` (red) and `Break` (green).

---

## Task 3: Implement Timer Logic in JavaScript
1. **Define Variables**
   - `timeLeft` (in seconds, default: `25 * 60`).
   - `timer` (stores the interval ID).
   - `isRunning` (Boolean to track state).

2. **Create Update Function**
   - Convert seconds to `MM:SS` format.
   - Update the display dynamically.

3. **Start Timer Function**
   - Use `setInterval()` to decrease `timeLeft` every second.
   - Update the UI.

4. **Pause Timer Function**
   - Use `clearInterval()` to stop countdown.

5. **Reset Timer Function**
   - Stop countdown and restore `25:00`.

---
